# 发现 APP - Flutter 社交分享应用

这是一个类似小红书的社交分享应用UI实现。

## 功能特点

- ✅ 首页展示（带Banner和分类标签）
- ✅ 帖子卡片展示
- ✅ 底部导航栏
- ✅ 用户信息展示
- ✅ 互动数据（浏览、评论、点赞）

## 项目结构

```
lib/
├── main.dart                 # 应用入口和主导航
├── models/
│   └── post.dart            # 帖子数据模型
├── screens/
│   └── home_screen.dart     # 首页
└── widgets/
    ├── banner_card.dart     # Banner卡片组件
    ├── category_tabs.dart   # 分类标签组件
    └── post_card.dart       # 帖子卡片组件
```

## 运行项目

1. 确保已安装 Flutter SDK
2. 在项目根目录运行：

```bash
flutter pub get
flutter run
```

## 已实现的UI组件

### 1. Banner卡片
- 绿色渐变背景
- 大标题"狐书-发现"
- 副标题"每一种兴趣"
- 装饰性圆点元素

### 2. 分类标签
- 横向滚动
- 包含：推荐、二次元、旅行、生活、绘画、摄影
- 选中状态带下划线

### 3. 帖子卡片
- 用户头像和昵称
- 发布时间
- 文字内容
- 图片网格（支持1-3张图片）
- 互动数据（浏览、评论、点赞）

### 4. 底部导航
- 发现、话题、发布（+按钮）、消息、我
- 发布按钮为绿色圆形
